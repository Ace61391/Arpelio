# Arpelio

Interactive fingering charts & assessments for every band instrument.

**Live:** [arpelio.com](https://arpelio.com)

## What is this?

Arpelio is a web app for music teachers and students. It provides:
- Interactive fingering chart reference for 14 band instruments
- Auto-graded digital quizzes
- Flashcard study mode
- Printable worksheet generation (coming soon)

## Tech Stack

- **Framework:** Next.js 14 (React)
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **Data:** 14 verified instrument JSON files (484 notes, 6,412 QC checks)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
